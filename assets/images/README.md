# Assets Folder

Place your SPECTRE logo file here as `logo.png`.

## Logo Requirements

- **File name**: `logo.png`
- **Format**: PNG with transparency
- **Recommended size**: 500x500px or larger
- **Background**: Transparent (no background)
- **Style**: Dark, mysterious, luxury aesthetic

## Usage

The logo is used in multiple places:
- Profile image (circular, 120px)
- Loading screen (120px)
- Footer logo (60px, grayscale)
- Open Graph/social media preview

Place your official SPECTRE logo in this folder and rename it to `logo.png`.
